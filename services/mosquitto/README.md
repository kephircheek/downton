# Mosquitto (MQTT Broker)

## Секреты
```
MQTT_ENDPOINT          сhar*  IP адресс MQTT-сервера, например "192.168.1.1"
MQTT_PORT              int    Номер порта, который слушает MQTT-сервер. По умолчанию 1883
MQTT_USER              char*  Имя пользователя
MQTT_PASSWORD          char*  Пароль пользователя
MQTT_PAYLOAD_MAX_SIZE  int    Лимит на размер сообщения в байтах, например 2048.   
```
